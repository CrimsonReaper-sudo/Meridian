# Meridian

A macOS menu-bar app for scheduling, focus sessions, and countdowns — with home-screen widgets.

## Download

Grab the latest `.dmg` from [**Releases**](../../releases/latest).

## Install

1. Open the downloaded `.dmg` and drag **Meridian** onto **Applications**.
2. On first launch macOS will warn that the developer can't be verified — the app is code-signed but not notarized (Apple's notarization requires a paid developer account).
   - Open **System Settings → Privacy & Security**, scroll to the bottom, and click **Open Anyway**.
   - Or right-click the app in Applications → **Open** → **Open**.

   You only have to do this once.

## Requirements

- macOS 26 (Tahoe) or later
- Apple Silicon or Intel (universal build)

## Features

- Menu-bar clock with your schedule at a glance
- Time-blocked daily schedule with alarms
- Pomodoro focus timer with saved presets
- Countdown timers
- Home-screen widgets: current session, pomodoro ring, reminders
- Optional Google Calendar and Gmail integration
- AI-assisted notes
